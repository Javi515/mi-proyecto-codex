# Orquídea Cocina

Sitio web estático para el restaurante moderno "Orquídea Cocina". Incluye páginas de inicio, menú, reservas, sobre nosotros y contacto, con estilos y scripts personalizados.

## Cómo verlo
- Abre el archivo `index.html` directamente en tu navegador favorito.
- Opcional: publica la carpeta del proyecto en GitHub Pages o cualquier hosting estático para habilitar la navegación completa.

## Estructura del proyecto
```
.
├── index.html
├── menu.html
├── reservas.html
├── sobre.html
├── contacto.html
├── assets
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── main.js
│   └── img
│       └── .gitkeep
└── README.md
```

## Créditos de recursos
Las imágenes utilizadas provienen de [Unsplash](https://unsplash.com/) mediante enlaces directos.
